使用Google PageSpeed Tools优化网站性能

4月 21, 2014 发布在 电脑网络

在做网页的时候，我们总是希望，我们的网站能够能够快一点，再快一点，但是如果没有一个好的工具，我们就很难知道网站到底是不是很快，也不知道怎么去让它快一点。强大的google给我们提供的PageSpeed Tools强大的工具。可以在线分析并给出解决办法。
其中包括十项检查：

清除首屏内容中阻止呈现的Javascrlpt和css

使用浏览器缓存

缩短服务器响应时间

启用压缩

压缩CSS

优化图片

按优先级排列可见内容

压缩 JavaScript

压缩 HTML

避免使用目标网页重定向
