Hexo笔记

10月 15, 2015

之前折腾过一些日子的Wordpress，现在开始搞 Hexo，在这里做一些折腾记录

hexo命令

简写
hexo n == hexo new #新建文章

hexo p == hexo publish

hexo g == hexo generate #生成

hexo s == hexo server #启动服务预览

hexo d == hexo deploy #部署

监视文件变动
hexo generate #使使用hexo生成静态文件

hexo generate --watch #监视文件变动
